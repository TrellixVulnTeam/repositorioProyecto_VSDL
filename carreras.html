Fullstack

1° HTML

HTML es el lenguaje sobre el que está construida toda la web y su estándar HTML5 la revolucionó por completo haciendo del navegador el programa más usado en el mundo.

¿Qué aprenderás?

Crear documentos HTML. Crear páginas web bien estructuradas. Conocer las últimas características de HTML5. Mantenerse actualizado en el estándar HTML.

¿Qué conocimientos previos necesitas?

Saber navegar por internet y tener curiosidad por cómo funciona la web. Este curso es desde cero y te inicia en el desarrollo web.

2° CSS

CSS es un lenguaje creado para diseñar la web, cuando ves una web hermosa, con animaciones y efectos, estos se crearon con CSS. Este curso te enseñará desde cero a escribir código CSS con buenas prácticas e iniciar tu carrera como diseñador web.

¿Qué aprenderás?

Fundamentos, sintaxis y buenas prácticas de CSS. Selectores, pseudoclases, modelo de caja. Manejo del texto y del color

¿Qué conocimientos previos necesitas?

HTML 5

3° javascript

JavaScript es el único lenguaje que te ofrece utilizar tus conocimientos tanto en el frontend como en el backend. Aprende las bases de este lenguaje y así estarás encaminado a poder decidir el camino que más te guste o tomar ambos y convertirte en un Desarrollador FullStack.

¿Qué aprenderás?

Las bases y particularidades de JS. Escribir código óptimo y reutilizable. Manipular datos y colecciones de datos. Aplicar lógica de programación en JS.

¿Qué conocimientos previos necesitas?

HTML 5 Y CSS

4° Firebase

Firebase es la solución perfecta para agregarle backend como servicio a tu aplicación. En este curso vas a crear web apps y vas a ver cómo guardar y recuperar datos de la nube. Aprendé, además, a subir archivos a la web, a autenticar con Google, Facebook y Twitter, incorporá reglas de seguridad a tu base de datos para poder tener secciones privadas a las que podrán acceder solo usuarios registrados.

¿Qué aprenderás?

A lo largo del curso vas a desarrollar dos proyectos desde cero: Un chat real time con vanilla JavaScript y una app para subir imágenes al estilo Instagram con Firebase.​​

¿Qué conocimientos previos necesitas?

HTML5, CSS y Javascript
